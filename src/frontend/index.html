fusebox-front-and-back-end

Welcome to fusebox-front-and-back-end

This repository is a starting boilerplate for a typescript app using typescript services and the formidable fusebox v4 bundler.

Quick start

get the code

in a console type

git clone https://github.com/vegarringdal/fusebox-front-and-backend.git myapp
cd myapp
npm install

Then start the develoment web server

npm run debug

Then open this page your favorite browser http://localhost/

Features

static content

This file is in src\frontend\index.html

Editing the file will refresh the page automatically.

Editing the file src\frontend\index.css with also refresh the page automatically.

dynamic content

server content

Finally this boilerplate contains a small backend API.

It contains a traditionnal addition service

http://localhost/api/add?a=111&b=222
